Visual Odometry based Relocalization using ORB feature descriptor

The goal of this research is to develop a computationally inexpensive relocalization module for a Visual SLAM system. The module requires a single camera (Kinect in this case), coupled with computer vision techniques to quickly estimate it's position and orientation in the environment.

ORB feature descriptor is used to identify keypoints in a scene and generate a best match from the images in the map. Once, this is done, perspective transform is applied to the current image and the matched image, to generate a relative pose of the camera.

After accounting for pose correction, the depth data from kinect is used to estimate the distance, thus, completing relocalization.

The module can help identify the best matching image within 0.6 seconds among 1000 candidates.
